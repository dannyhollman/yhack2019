    <article class="markdown-body entry-content p-3 p-md-6" itemprop="text"><h1><a id="user-content-selenium-client-driver" class="anchor" aria-hidden="true" href="#selenium-client-driver"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Selenium Client Driver</h1>
<a name="user-content-introduction"></a>
<h2><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Introduction</h2>
<p>Python language bindings for Selenium WebDriver.</p>
<p>The selenium package is used to automate web browser interaction from Python.</p>

<a name="user-content-installing"></a>
<h2><a id="user-content-installing" class="anchor" aria-hidden="true" href="#installing"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installing</h2>
<p>If you have <a href="https://pip.pypa.io/" rel="nofollow">pip</a> on your system, you can simply install or upgrade the Python bindings:</p>
<pre>pip install -U selenium
</pre>
<p>Alternately, you can download the source distribution from <a href="https://pypi.org/project/selenium/#files" rel="nofollow">PyPI</a> (e.g. selenium-4.0.0a3.tar.gz), unarchive it, and run:</p>
<pre>python setup.py install
</pre>
<p>Note: You may want to consider using <a href="http://www.virtualenv.org/" rel="nofollow">virtualenv</a> to create isolated Python environments.</p>
<a name="user-content-drivers"></a>
<h2><a id="user-content-drivers" class="anchor" aria-hidden="true" href="#drivers"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Drivers</h2>
<p>Selenium requires a driver to interface with the chosen browser. Firefox,
for example, requires <a href="https://github.com/mozilla/geckodriver/releases">geckodriver</a>, which needs to be installed before the below examples can be run. Make sure it's in your PATH, e. g., place it in /usr/bin or /usr/local/bin.</p>
<p>Failure to observe this step will give you an error selenium.common.exceptions.WebDriverException: Message: 'geckodriver' executable needs to be in PATH.</p>
<p>Other supported browsers will have their own drivers available. Links to some of the more popular browser drivers follow.</p>

<h2><a id="user-content-example-2" class="anchor" aria-hidden="true" href="#example-2"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Example 2:</h2>
<p>Selenium WebDriver is often used as a basis for testing web applications.  Here is a simple example using Python's standard <a href="http://docs.python.org/3/library/unittest.html" rel="nofollow">unittest</a> library:</p>
<div class="highlight highlight-source-python"><pre><span class="pl-k">import</span> unittest

<span class="pl-k">class</span> <span class="pl-en">GoogleTestCase</span>(<span class="pl-e">unittest</span>.<span class="pl-e">TestCase</span>):

    <span class="pl-k">def</span> <span class="pl-en">setUp</span>(<span class="pl-smi"><span class="pl-smi">self</span></span>):
        <span class="pl-c1">self</span>.browser <span class="pl-k">=</span> webdriver.Firefox()
        <span class="pl-c1">self</span>.addCleanup(<span class="pl-c1">self</span>.browser.quit)

    <span class="pl-k">def</span> <span class="pl-en">testPageTitle</span>(<span class="pl-smi"><span class="pl-smi">self</span></span>):
        <span class="pl-c1">self</span>.browser.get(<span class="pl-s"><span class="pl-pds">'</span>http://www.google.com<span class="pl-pds">'</span></span>)
        <span class="pl-c1">self</span>.assertIn(<span class="pl-s"><span class="pl-pds">'</span>Google<span class="pl-pds">'</span></span>, <span class="pl-c1">self</span>.browser.title)

<span class="pl-k">if</span> <span class="pl-c1">__name__</span> <span class="pl-k">==</span> <span class="pl-s"><span class="pl-pds">'</span>__main__<span class="pl-pds">'</span></span>:
    unittest.main(<span class="pl-v">verbosity</span><span class="pl-k">=</span><span class="pl-c1">2</span>)</pre></div>
<a name="user-content-selenium-server-optional"></a>
<h2><a id="user-content-selenium-server-optional" class="anchor" aria-hidden="true" href="#selenium-server-optional"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Selenium Server (optional)</h2>
<p>For normal WebDriver scripts (non-Remote), the Java server is not needed.</p>
<p>However, to use Selenium Webdriver Remote or the legacy Selenium API (Selenium-RC), you need to also run the Selenium server.  The server requires a Java Runtime Environment (JRE).</p>
<p>Download the server separately, from: <a href="http://selenium-release.storage.googleapis.com/4.0/selenium-server-standalone-4.0.0.jar" rel="nofollow">http://selenium-release.storage.googleapis.com/4.0/selenium-server-standalone-4.0.0.jar</a></p>
<p>Run the server from the command line:</p>
<pre>java -jar selenium-server-standalone-3.141.0.jar
</pre>
<p>Then run your Python client scripts.</p>
